## 📖 About This Repository

This repository contains the **complete frontend implementation** of **LUMEN**, a web-based cybersecurity analysis platform designed to help users upload system logs and automatically detect potential security risks.

The focus of this project is to deliver a **modern, interactive, and user-friendly interface** with dynamic themes and smooth user experience.

## ✨ Key Features

* 🎨 **Mood-Based UI System (8 Themes)**
* 🏠 **Interactive Home Page**
* 🔐 **Authentication System (Login & Registration)**
* 🔑 **Password Recovery with OTP Verification**
* 📂 **Drag & Drop Log Upload System**
* 📊 **Risk Analysis Dashboard (Charts & Insights)**
* 📋 **Copy-Paste Security Commands**
* 🛠️ **Admin Panel with Management Controls**
* 💎 **Premium Support Request System**
