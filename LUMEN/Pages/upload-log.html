How to Upload Logs from Your Laptop

1

Locate Your Log File

Log files are usually stored in these locations:

📁 Windows: C:\Windows\System32\winevt\Logs\ 📁 Linux: /var/log/ 📁 Apache: /var/log/apache2/access.log 📁 Nginx: /var/log/nginx/access.log
2

Choose Upload Method

Browse Files Click to select from your computer
Drag & Drop Drag file into the upload area
3

Supported File Formats

.log .txt .json .csv

Maximum file size: 10MB

4

Click "Analyze Log"

After uploading, click the analyze button to get your security report

Quick Tips

  • Make sure your log file is in plain text format
  • For best results, use Apache or Nginx access logs
  • You can export Windows Event Logs as .txt files
  • Try the Demo button to see how it works
